# Kynd — Trusted house help app

Vite + React + Tailwind landing page wrapped with Capacitor for Android/iOS.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Capacitor (mobile)
```bash
# first time
npm run build
npm run cap:add:android   # or cap:add:ios (Mac only)
npx cap sync

# open native IDE
npm run cap:open:android
npm run cap:open:ios
```
